# FA360 App Icons

Este diretório deve conter os ícones da aplicação PWA.

## Ícones necessários

Gere os seguintes ícones a partir do logo FA360:

- `icon-16x16.png` - Favicon pequeno
- `icon-32x32.png` - Favicon padrão
- `icon-72x72.png` - Android Chrome
- `icon-96x96.png` - Android Chrome / Shortcuts
- `icon-128x128.png` - Chrome Web Store
- `icon-144x144.png` - MS Tile
- `icon-152x152.png` - Apple Touch Icon
- `icon-192x192.png` - Android Chrome / PWA
- `icon-384x384.png` - Android Chrome
- `icon-512x512.png` - PWA Splash

### Ícones de atalho
- `shortcut-project.png` - 96x96 - Atalho para novos projetos
- `shortcut-proposal.png` - 96x96 - Atalho para propostas
- `shortcut-calendar.png` - 96x96 - Atalho para calendário

## Ferramentas recomendadas

- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/nicmut/pwa-asset-generator)
- [Maskable.app](https://maskable.app/) - Para testar ícones maskable

## Especificações

- Formato: PNG
- Fundo: Transparente ou com cor sólida (#4f46e5 recomendado)
- Safe zone para maskable: 80% do centro
