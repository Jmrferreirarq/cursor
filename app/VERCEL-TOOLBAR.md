# Esconder a Vercel Toolbar

A barra azul à direita é a **Vercel Toolbar**, injetada quando o site está no Vercel.

## Opções para esconder

### 1. Dashboard do Vercel (recomendado)
1. Acede a [vercel.com](https://vercel.com) → teu projeto **cursor**
2. **Settings** → **General**
3. Encontra **Vercel Toolbar**
4. Em **Production**, escolhe **Off**

### 2. Extensão Chrome
1. Clica no ícone da extensão Vercel (barra de favoritos)
2. Abre **Preferences**
3. Ativa **Start Hidden** — a toolbar começa oculta
4. Usa `Ctrl + .` para mostrar quando precisares

### 3. Na sessão atual
- Clica na toolbar → menu → **Disable for Session**
